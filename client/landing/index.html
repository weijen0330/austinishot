Why Lynx?



The Problem

With 205 billion messages being sent per day, how do we create a service that allows users to easily archive and retrieve links that they send over popular messaging services, and minimize time spent scrolling through message history?

The Solution

The Lynx web-app automatically archives links that users send over popular messaging services. Lynx recognizes that links shared over messaging holds value to the user and should not be lost in their message history
